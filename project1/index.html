Simple HTTP Server

This simple HTTP server utilizes the C++ programming language and BDS sockets. For more information about the operation of this program, please see the README!

Functionality

This server, while basic, fufills a decent range of functionality. Features include:

  • Case-insensative filenames
  • Serving of basic filetypes (eg .html, .jpg, .gif) within the browser
  • Ability to send large files, exceeding 100 Mb
